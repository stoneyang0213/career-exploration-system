配置OpenAI API密钥以启用AI专业报告功能

OpenAI API设置

您的API密钥将安全存储在本地，不会上传到服务器

如何获取OpenAI API密钥

1

注册OpenAI账号

访问 OpenAI Platform 并注册账号

2

创建API密钥

在API Keys页面点击"Create new secret key"创建新的API密钥

3

复制密钥

复制生成的API密钥（以sk-开头），粘贴到上面的输入框中

4

保存设置

点击"保存设置"按钮，然后就可以使用AI专业报告功能了

费用说明

OpenAI API按使用量收费，每次生成报告约需$0.01-0.05
新用户可获得免费额度，足够生成多份报告
您可以在OpenAI控制台查看详细的使用情况和费用
